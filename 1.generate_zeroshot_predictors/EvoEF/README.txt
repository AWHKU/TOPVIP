# Running EvoEF #
We generate the EvoEF DDG scores using [EvoEF2](https://github.com/tommyhuangthu/EvoEF2.git).

The input files for the sripts are provided at *data/zeroshot_prediction/EvoEF*.

An example R scripts (SpCas9_makeEvoEF_zeroshot_Prep_summarize.R) to phrase the evoEF output were provided afor the SpCas9 dataset.
